# Before doing all this make sure these are installed: stow, git


- run this to give permission to run the script  

```bash
chmod +x $HOME/.dotfiles/.bin/sync-dotfiles
```

- verify permission

```bash
ls -l $HOME/.dotfiles/.bin/sync-dotfiles
```
